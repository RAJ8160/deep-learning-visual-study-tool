Models

Three Architectures,
One Dataset

Watch how complexity scales from a single linear layer to deep convolutional feature extraction.

🔵
Perceptron
Single-Layer · Linear

784 inputs → 10 outputs. No hidden layers, no spatial awareness. The foundation everything else builds on.

~92%
🟡
ANN · MLP
Multi-Layer · Non-linear

Hidden layers (128 + 64) with ReLU and Dropout. Learns curved decision boundaries — far more expressive.

~97%
🟢
CNN
Convolutional · Spatial

32 + 64 learned filters slide across the 2D image. Detects edges → curves → shapes. Highest accuracy because it respects image structure.

~99%
